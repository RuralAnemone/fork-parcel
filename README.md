## Supported services

International:

- DHL
- GLS
- UPS

United Kingdom:

- DPD UK
- Evri

Europe:

- Belpost (BY)
- Magyar Posta (HU)
- Packeta
- Poczta Polska (PL)
- Poste Italiane (IT)
- Sameday Bulgaria
- Sameday Hungary
- Sameday Romania

Asia:

- SPX Thailand
